### 시작하기

```bash
npm run dev
npm run build-storybook
npm run storybook
```

### 로컬 서버 주소

- dev
  - http://localhost:3100
- storybook
  - http://localhost:6006

### .env.local

```bash
NEXT_PUBLIC_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY
NEXT_PUBLIC_API_URL

LIVEKIT_API_KEY
LIVEKIT_API_SECRET
NEXT_PUBLIC_LIVEKIT_URL

SENTRY_AUTH_TOKEN
```
